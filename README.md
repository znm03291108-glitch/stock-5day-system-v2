# 5日线强势股交易纪律系统 V3.7：新股异常过滤 + 有效5日线确认版

在 V3.6.9 基础上增加强过滤规则，解决“001399 涨幅315%、MA5为空却高分”的问题。

新增/修复：
- 自动排除 N/C/U/W 新股或特殊上市标识
- 自动排除涨幅超过30%的异常波动票
- 自动排除 MA5 缺失或无效股票
- 自动标记“有效5日线：是/否”
- 无有效MA5不生成实盘交易计划
- 异常候选最高分限制为59分
- 股票卡片增加“有效5日线 / 风险过滤”标签
- 新增筛选按钮：有效5日线、风险过滤、MA5缺失
- 新增 /api/filter_status
- 保留手机展示优化、财报稳定、大盘情绪、题材深度分析

核心原则：
没有有效5日线，就不参与5日线交易纪律评分。
