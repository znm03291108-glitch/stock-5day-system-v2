# 5日线强势股交易纪律系统 V3.7.3.6：观察池前空结果提示版

本版修复 V3.7.3.5 仍未显示空结果提示的问题。

原因：
- 前几版尝试把提示插入“智能股票卡片”区域
- 但实际页面 DOM 结构中，股票卡区域可能被隐藏/清空
- 所以提示找不到稳定插入位置

本版方案：
- 直接把空结果提示固定插入到“观察池”上方
- 只要主列表没有有效5日线股票，就一定显示提示
- 不再依赖智能股票卡片区域结构
- 不再依赖是否还能找到被隐藏股票卡

新增：
- 今日暂无有效5日线股票提示
- 有效5日线数量
- 已过滤/降级数量
- 总候选参考数量
- 过滤原因汇总
- 查看已过滤按钮
- 重新筛选按钮

新增接口：
- /api/watchpool_empty_status

检查：
- /api/health 应显示 3.7.3.6-watchpool-before-empty
- /api/watchpool_empty_status
- 页面强制刷新：/?v=3736
