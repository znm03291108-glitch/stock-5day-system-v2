# 5日线强势股交易纪律系统 V3.7.3：主结果净化 + 对象显示修复版

本版重点是让结果更干净、更可信，不再把新股、ST、MA5缺失、异常涨幅票混入主结果。

修复/新增：
- 修复“操作建议：[object Object]”
- 默认主结果只展示“有效5日线”股票
- N/C/U/W 新股或特殊上市标识自动过滤
- ST/*ST/退市风险自动过滤
- MA5缺失或无效自动过滤
- 涨幅超过30%自动过滤
- 换手率超过35%标记高风险
- 成交额不足1亿过滤
- 风险过滤股票不生成5日线交易计划
- 无效候选降级为“快速候选/已过滤”
- 新增“查看已过滤”
- 新增“有效候选 / 已过滤 / 总候选”统计
- 新增 /api/clean_filter_status
- 保留做T纪律、财报解释、大盘情绪、交易日状态、观察池

核心原则：
宁可少给几只干净票，也不要把无效5日线和异常票放进主结果。
