# 5日线强势股交易纪律系统 V3.7.3.3：DOM最终净化版

本版是针对前端旧渲染函数覆盖问题的最终兜底修复。

修复：
- 右上角标签仍显示 undefined
- “全部”仍显示 MA5 缺失股票
- 旧渲染函数把风险票重新渲染到主列表
- 新股、ST、异常涨幅仍混入主股票卡

新增：
- DOM层最终净化
- 自动隐藏主列表中的 MA5缺失卡片
- 自动隐藏 N/C/U/W 新股
- 自动隐藏 ST/*ST/退市风险
- 自动隐藏涨幅超过30%的异常卡片
- undefined 标签自动修正
- 新增 DOM净化统计提示
- 新增 /api/dom_final_clean_status

使用规则：
- 点击“全部”= 只看有效5日线
- 点击“有效5日线”= 只看有效5日线
- 点击“查看已过滤 / 风险过滤 / MA5缺失”= 可复核风险票
- 风险票不参与5日线交易计划
