修复：

- 点击“全部”仍然显示 MA5 缺失、新股、异常票
- 主股票卡仍混入无效5日线候选

规则：
- “全部”按钮 = 全部有效5日线，不是全部候选

- MA5 缺失/无效只进入“查看已过滤”

- 风险过滤股票不生成实盘交易计划
- 标签为空/undefined 自动显示“已过滤”或“有效5日线”
