# 5日线强势股交易纪律系统 V3.6.5：大盘情绪联动版

在 V3.6.4 基础上新增：
- 上证指数、深成指、创业板指数涨跌
- 上涨家数 / 下跌家数
- 涨停家数 / 跌停家数
- 大盘情绪分
- 大盘状态：强势、偏强、中性、偏弱、弱势
- 仓位系数
- 大盘弱时自动降低仓位建议
- 大盘强时提高优质安全买点观察等级
- 新增 /api/market_sentiment 接口
- 股票卡片新增“大盘情绪联动”提示

使用：
1. 点击“今日智能筛选”
2. 系统自动读取大盘情绪
3. 个股卡片会结合大盘强弱调整仓位建议
4. 大盘弱时不追高，大盘强时也必须按5日线纪律分批

风险提示：大盘情绪只是仓位辅助，不是买卖指令。
