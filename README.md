# 5日线强势股交易纪律系统 V3.6.8：财报接口稳定增强版

本版重点修复：财报取不到时，不再误判为“谨慎低分”。

新增/修复：
- 财报接口多源兜底
- AKShare 不同字段名自动兼容
- 财报取不到时显示“未取得”，不参与评分
- 财报失败不影响单股分析
- 增加“仅技术面模式”
- 增加“财报数据置信度”
- 增加“财报缺失原因”
- 保留 V3.6.7.2 单股分析修复
- 保留交易日状态识别、大盘情绪、题材深度分析、交易计划

核心原则：
财报缺失 ≠ 财报差。
财报缺失时，系统按技术面、5日线、大盘情绪和题材持续性处理。
