# 5日线强势股交易纪律系统 V3.7.3.5：空结果提示强制显示版

本版修复 V3.7.3.4 的问题：
- 版本已生效，但空结果提示框没有显示
- 原因：上一版依赖股票卡元素，如果无效卡被隐藏或清空，就无法触发空结果提示

新增修复：
- 空结果提示不再依赖股票卡元素
- 主列表没有有效卡片时强制显示提示
- 已过滤原因可从 DOM、已过滤数组、本地统计中兜底读取
- 点击今日智能筛选、手动分析、自动连续分析后会多次检查并显示空结果
- 滚动到底部也能看到提示，不再只显示观察池和数据管理

新增接口：
- /api/force_empty_summary_status

检查：
- /api/health 应显示 3.7.3.5-force-empty-summary
- /api/force_empty_summary_status
- 页面强制刷新：/?v=3735
