# 5日线强势股交易纪律系统 V3.6.7：财报结果中文解释版

在 V3.6.6 基础上新增：
- 财报/公告原始 JSON 自动转换成中文解释
- 显示报告期、净利润同比、营收同比、毛利率、ROE
- 自动提取积极点、风险点、中性信息
- 自动识别“净利润增长但营收下降”等特殊情况
- 生成财报评级：积极、偏积极、中性、谨慎
- 生成财报分数
- 新增 /api/finance_explain 接口
- “测试真实数据”升级为“测试财报解释”
- 股票卡片可展示财报中文解释

使用：
1. 输入股票代码
2. 点击“测试财报解释”
3. 查看中文财报卡片
4. 展开原始 JSON 可查看完整数据
5. 实盘前仍需核对公告原文

风险提示：财报解释只用于辅助理解，不构成投资建议。
