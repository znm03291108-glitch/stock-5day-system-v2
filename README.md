# 5日线强势股交易纪律系统 V3.7.3.4：空结果提示 + 已过滤原因汇总版

本版是在 V3.7.3.3 DOM 最终净化版基础上继续增强体验。

解决的问题：
- 主股票卡为空时，用户不知道是没数据还是系统坏了
- 无法看到今天到底过滤了多少只股票
- 无法知道过滤原因主要是什么
- 无效候选被隐藏后缺少复核入口

新增功能：
- 主股票卡为空时显示明确提示
- 显示有效5日线数量
- 显示已过滤数量
- 显示总候选数量
- 显示过滤原因汇总：
  - MA5缺失或无效
  - 新股/特殊上市标识
  - ST/退市风险
  - 涨幅异常超过30%
  - 换手率过高
  - 其他
- 一键查看已过滤
- 一键重新筛选
- 保留 DOM 最终净化，继续隐藏无效5日线股票

新增接口：
- /api/empty_filter_summary_status

检查：
- /api/health 应显示 3.7.3.4-empty-filter-summary
- /api/empty_filter_summary_status 测试空结果和过滤汇总
- 页面强制刷新：/?v=3734

使用建议：
- 如果提示“暂无有效5日线”，说明当天候选质量不够，不建议强行交易。
- 可以点击“查看已过滤”复核，但过滤票不参与5日线交易计划。
- 周末/非交易日数据多为复盘参考，不按实时盘中买卖判断。

风险提示：
本系统为复盘和交易纪律辅助工具，不构成投资建议。
