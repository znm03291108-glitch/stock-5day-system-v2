# 5日线强势股交易纪律系统 V3.6.3：热门题材可点击版

在 V3.6.2 基础上新增：
- 热门题材/概念支持点击
- 点击题材后自动读取该题材下股票
- 题材股票自动显示到“智能股票卡片”
- 题材股票继续过滤 688、北交所等
- 题材股票可继续点“自动连续分析”做5日线深度判断
- 新增 /api/theme_stocks 接口
- 保留最新财报修正、技术优先、实盘交易计划

使用：
1. 点击“今日智能筛选”
2. 到“热门题材/概念”区域
3. 点一个题材，例如“玻璃基板”
4. 系统显示该题材下股票
5. 再点“自动连续分析”确认5日线

风险提示：题材只是热度，不是买点；最终仍按5日线纪律执行。
