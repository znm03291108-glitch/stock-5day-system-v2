# 5日线强势股交易纪律系统 V3.6.4：题材点击后自动深度分析版

在 V3.6.3 基础上新增：
- 点击热门题材后，自动读取题材股票
- 可选择自动深度分析前 3 / 6 / 10 只
- 深度分析后直接显示 MA5、距5日线、交易计划、财报公告
- 深度失败的股票保留快速候选
- 可继续点击“自动连续分析”补全剩余股票
- /api/theme_stocks 支持 auto_deep 和 deep_limit 参数

使用：
1. 点“今日智能筛选”
2. 勾选“点击题材后自动深度分析”
3. 选择自动分析数量，推荐 6 只
4. 点击一个题材
5. 系统自动显示题材股票并分析前几只
6. 再点“自动连续分析”可补全剩余股票

风险提示：题材只是热度，不是买点；最终仍按5日线纪律执行。
