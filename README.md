# 5日线强势股交易纪律系统 V3.6.6：交易日状态识别版

在 V3.6.5 基础上新增：
- 自动识别当前是否周末
- 自动识别盘前、集合竞价、早盘交易中、午间休市、午后交易中、尾盘观察、2:50纪律执行、收盘后
- 页面新增“交易日状态”
- 自动提示数据用途：复盘参考 / 盘前准备 / 盘中快照 / 尾盘检查 / 实盘纪律
- 周末提示：数据多为上一交易日收盘数据，只适合复盘
- 尾盘提示：重点执行5日线纪律
- 新增 /api/trading_status 接口
- 保留大盘情绪联动、题材点击后自动深度分析、最新财报修正

使用：
1. 点击“今日智能筛选”
2. 先看“交易日状态”
3. 如果是周末或收盘后，只用于复盘
4. 如果是盘中，结合大盘情绪和5日线纪律执行
5. 2:50窗口重点看减仓、止损、是否留仓

风险提示：交易日状态识别不能完全覆盖法定节假日，实盘前仍需人工核对交易所日历。
